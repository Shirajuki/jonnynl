Questions

  1. The server in the client-server architecture acts as a host that waits for requests from clients and responds to them respectively according to the service the server provide.
  2. The client in the client-server architecture acts as server requesters. It requests and recieves a servers service. In short, it's like exchanging of messages between them. The client sends a request, and the server returns a response.
  3. An IP-address is a unique set of numbers that identifies an IT device connected to a network. An IP-address is then defines in either IPv4 format or IPv6, and is usually used to identificate the device/host.
  4. DNS, acronym of Domain Name System, is a directory of domains used to translate a domain name into it's matching IP-address to make it easier for us to access services based on IP-addresses.
  5. The difference between the Internet and World Wide Web is that the Internet is a global network of networks, while the WWW is just a collection of web pages that can be accessed via the Internet following the http protocol (a kind of language used on the internet to communicate and transfer data).
  6. HTML, acronym for Hyper Text Markup Language, is a markup language (a computer language that uses tags to define it's elements) for creating web pages in the Web. It's usually used together with technologies such as CSS and JS.
