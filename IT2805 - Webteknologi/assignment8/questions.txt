2. How would you create a blue circle using the Canvas element? Explain with words or code.
- Canvas drawing circle is given `ctx.arc(x, y, radius, 0, Math.PI*2);`
so to create a blue circle on canvas with id="myCanvas" at x=100, y=100 and radius=50 we write:
```
const canvas = document.getElementById('myCanvas');
const context = canvas.getContext('2d');
context.beginPath();
context.fillStyle = "blue";
context.arc(100, 75, 50, 0, Math.PI*2);
context.fill();
```

3. What is a more terse (also known as shorter) way of writing the loop in the following using
the Array.filter method?
```javascript
const countries = ['Norway', 'Sweden', 'Denmark', 'New Zealand'];
const countriesStartingWithN = [];
for (let i = 0; i < countries.length; i++) {
  let country = countries[i];
  if (country.startsWith('N')) {
    countriesStartingWithN.push(country);
  }
}
```
- A terser/shorter way using Array.filter method:
```
const countries = ['Norway', 'Sweden', 'Denmark', 'New Zealand'];
const countriesStartingWithN = countries.filter(country => country[0] == "N");
```
